# Crosshair - Customizable Screen Crosshair

A lightweight and customizable crosshair overlay addon for World of Warcraft. Displays a simple two-line crosshair in the center of your screen with full customization options.

## Key Features

✅ **Simple Design** - Clean two-line crosshair (horizontal and vertical lines)  
✅ **Fully Customizable** - Adjust size (5-100), color (RGB), and position (X/Y offsets)  
✅ **Easy Configuration** - Access settings via `/crosshair` or `/ch` command  
✅ **Multi-Language** - Supports English, German, Spanish, and French  
✅ **Persistent Settings** - All preferences saved automatically  

## Quick Start

1. Extract to `Interface\AddOns\`
2. Type `/crosshair` in-game to open settings
3. Customize size, color, and position to your preference

Perfect for players who want a visual reference point in the center of their screen!
